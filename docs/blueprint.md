# **App Name**: MilkRoute

## Core Features:

- User Authentication: Register and login using Firebase Authentication (email/password or phone).
- Product Browsing: Browse milk products with details (name, price, image) from Firestore.
- Order Placement: Place orders online, selecting quantity and delivery address. 
- Payment Options: Choose payment method: Cash on Delivery (COD) or online payment (Razorpay or Google Pay).
- Order Tracking: Track order status (Pending, Delivered, Cancelled) stored in Firestore.
- Admin Panel: Admin panel to view orders, update delivery status, and manage products.
- Push Notifications: Alert users about delivery updates via Firebase Cloud Messaging (FCM).

## Style Guidelines:

- Primary color: Sky blue (#87CEEB), reminiscent of a clear morning sky and fresh milk.
- Background color: Light cream (#F8F8FF), providing a clean and soft backdrop.
- Accent color: Light Green (#90EE90) for calls to action, freshness.
- Body and headline font: 'PT Sans', a humanist sans-serif that's modern and friendly.
- Use line icons with rounded edges, symbolizing cleanliness and approachability.
- Clean and spacious layout with clear sections for browsing products and managing orders.
- Subtle transitions for order status updates and product loading.
